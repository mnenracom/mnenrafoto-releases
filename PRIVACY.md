# Политика конфиденциальности — МНЕНРАФОТО Alpha 2

## Обработка фотографий

- Все фотографии обрабатываются **локально** на компьютере пользователя.
- Программа **не отправляет** изображения на серверы правообладателя или третьих лиц.
- **Облачный конвертер** для HEIC или других форматов **не используется**.

## Телеметрия и аккаунты

- **Телеметрия** и **аналитика** не собираются.
- **Учётная запись** не требуется.

## Локально сохраняемые данные

На компьютере пользователя могут сохраняться:

- пользовательские пресеты печати и цвета;
- последний выбранный принтер и связанные настройки;
- состояние onboarding;
- выбранный язык интерфейса;
- память текущего и предыдущего заказа;
- пользовательские настройки режима «Фото на документы»;
- диагностический clean-print PNG, если включена соответствующая диагностика;
- ссылки на исходные локальные пути фотографий для восстановления заказов.

Данные хранятся в каталоге userData Windows для приложения Electron.

## Временные файлы

- Временные HEIC/JPEG файлы декодирования создаются в системной TEMP-папке и удаляются после использования.
- Временные raster/print файлы создаются для подготовки печати и удаляются по завершении операции или при очистке.

## Удаление данных

- Деинсталлятор Setup **по умолчанию не удаляет** userData, пресеты и память заказов.
- Пользователь может удалить сохранённые данные вручную через удаление каталога userData приложения.

## Сеть

- Программа не требует постоянного подключения к интернету для работы с фотографиями и печатью.
- HEIC декодируется через локальный helper и системный Windows decoder.

## Обращения в Issues

При сообщении об ошибке не прикладывайте чужие личные фотографии, документы или изображения с персональными данными. Для проверки лучше использовать нейтральные тестовые файлы.
